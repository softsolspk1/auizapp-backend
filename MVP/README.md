# Hilton Quiz App - MVP Demo

## 🎯 Project Overview

This is a comprehensive MVP (Minimum Viable Product) demonstration of the Hilton Quiz App - a medical knowledge competition platform designed specifically for healthcare professionals. The MVP showcases both the **Web Admin Panel** and **Mobile App** interfaces in HTML format, perfect for client presentations and demonstrations.

## 📱 What's Included

### 1. **Main Demo Page** (`index.html`)
- Project overview and feature highlights
- Direct links to Admin Panel and Mobile App demos
- Professional presentation layout

### 2. **Web Admin Panel** (`admin-panel.html`)
- **Dashboard**: Overview with key metrics, charts, and statistics
- **User Management**: Approve/activate doctor accounts, view user details
- **Category Management**: Create and manage quiz categories
- **Question Management**: Add, edit, and organize quiz questions
- **Analytics**: Comprehensive insights with interactive charts
- **Real-time Data**: Simulated live statistics and performance metrics

### 3. **Mobile App** (`mobile-app.html`)
- **Splash Screen**: Branded welcome screen with auto-navigation
- **Authentication**: Login, registration, and forgot password screens
- **Home Dashboard**: User stats, quick actions, and category selection
- **Quiz Interface**: Interactive quiz with timer and scoring
- **Results Screen**: Detailed performance analysis
- **Profile Management**: User information and statistics
- **Leaderboard**: Top performers ranking
- **Multiplayer Mode**: Room creation and joining interface

## 🚀 How to Use

### Quick Start
1. Open `index.html` in your web browser
2. Click on "Admin Panel Demo" or "Mobile App Demo"
3. Navigate through the different screens and features
4. Experience the full user journey

### Admin Panel Navigation
- Use the sidebar navigation to switch between sections
- Click on buttons to open modals and forms
- View interactive charts and statistics
- Experience the complete admin workflow

### Mobile App Navigation
- Start with the splash screen (auto-advances after 3 seconds)
- Navigate through login/registration screens
- Use the bottom navigation to switch between main sections
- Experience the quiz flow and results

### Single Player Mode
- **Correct Answer**: +10 points
- **Wrong Answer**: -5 points

### Multiplayer Mode
- **Correct Answer**: +20 points (double points)
- **Wrong Answer**: -5 points
